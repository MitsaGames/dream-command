# Установка

## Глобально (доступно из любого проекта)

Скопируйте файл команды в пользовательскую директорию команд:

```
~/.claude/commands/dream.md
```

На Windows:

```
%USERPROFILE%\.claude\commands\dream.md
```

## Только для текущего проекта

Скопируйте файл команды в директорию `.claude/commands/` внутри проекта:

```
<корень-проекта>/.claude/commands/dream.md
```

## Использование

| Команда | Область действия |
|---------|-----------------|
| `/dream` | Консолидация памяти текущего проекта |
| `/dream user` | Консолидация пользовательской памяти |
| `/dream all` | Оба уровня сразу |
